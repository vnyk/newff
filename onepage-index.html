Getting started with Python for DS(120 Minutes)

Installing Python and Python Editors (20 Minutes)

Python Basics (40 Minutes) ○ Basic Syntax and Data types ○ Data Structures (Lists, Sets, Tuples, Dictionaries) ○ High Performance Container Data Types - Collections ○ Datetime, Calendar, heapq ○ Iterators (itertools) and generators ○ pickle - Python object serialization, cpickle ○ Operators, Control Statements, User defined functions and classes

Data Import and Manipulation in Python (60 Minutes)
● NumPy with Python (30 Minutes) ○ Basic Array Operations, Comparison Operations and Value Testing ○ Vector and Matrix Mathematics ○ Generating Statistics, Numpy random numbers ○ Polynomial Mathematics ○ Numpy Array Broadcasting ● Pandas (30 Minutes) ○ Importing the Dataset, handling Excel/CSV Files ○ Using pandas Data Frames to solve complex tasks ○ Summarizing, Aggregation and Grouping Data using Apply operations ○ Descriptive Statistic and Pivot Table Summaries

Data Visualization and ml basics in Python (60 Minutes)t

Module 3: Data Visualization in Python (60 Minutes) ● Use Matplotlib and Seaborn for data visualizations ● Creating Line plot, Bar Chart, Pie Chart, Histogram, Scatter Plots and Contour Plots ● Use plotly for Interactive visualizations

Module 4: Basics of Machine Learning Models (60 Minutes) ● Supervised vs Unsupervised Learning, Discriminative vs Generative Algorithms ● Linear/Logistic Regression, K-Nearest Neighbors ● Support Vector Machines and Kernel Functions, Naive Bayes Classifier ● Clustering Techniques (K-Means Clustering) ● Decision Trees, Bagging Techniques and Random Forests ● Boosting Techniques (XGBoost and AdaBoost)

Putting everything together in Scikit (120 Minutes)

Module 5: Loss Functions, Optimization Techniques and Evaluation Metrics (60 Minutes) ● Bias vs Variance Tradeoff ● Objective/Loss Functions (MSE, Sigmoid, Softmax), Optimization Techniques (Gradient Descent and Stochastic Gradient Descent) ● L1 and L2 Regularisation ● Evaluation Metrics (accuracy, precision, recall, mse, mae) ● Model hyperparameters tuning using Cross Validation and Leave one out validation

Module 6: Putting everything together in Scikit (60 Minutes) ● Data Preprocessing (20 Minutes) ○ Missing Data Imputation, Handling Categorical Data ○ Splitting the Dataset into the Training set and Test set ○ Feature Extraction ○ Feature Scaling Techniques (Min-Max scaling, PCA whitening) ○ Dimensionality Reduction using Singular Value Decomposition (SVD) ● Model Fitting and Tuning (20 Minutes) ○ Model fit and predict functions ○ Model Selection, Cross-validation and Hyperparameter tuning ● Model Evaluation (20 Minutes) ○ Estimator score method ○ Scoring parameter ○ Metric functions

Neural Networks (60 Minutes)

● Basics of Neural Networks ● Single Hidden Layer Neural Networks and Backpropagation ● Multilayer Neural Networks and Multi-output Neural Networks ● Training Multilayer Neural Networks in Keras
